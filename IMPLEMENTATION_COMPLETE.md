## What You Can Do Now

### 1. Conduct Structured Interviews
```bash
# Start interactive interview
curl -X POST http://localhost:8001/interview/start \
  -H "Content-Type: application/json" \
  -d '{"user_id":"user123"}'

# Gemini generates adaptive questions based on answers
# After 5-7 questions, get personalized recommendation
```

### 2. Design Solutions with AI
```bash
# Submit a problem statement
curl -X POST http://localhost:8001/orchestration/start \
  -H "Content-Type: application/json" \
  -d '{"user_id":"user123","query":"Design an agent system for data analysis"}'

# Agents:
# 1. Supervisor analyzes with Gemini
# 2. Router decomposes with Gemini
# 3. RAG searches patterns with Gemini
# 4. Solution Builder designs architecture with Gemini
# 5. All results synthesized into recommendation
```

### 3. Deploy to Databricks
```bash
# After approval, deployer creates job in workspace
# Access Databricks → Workflows → Jobs
# See "orchestration-job-xxx" created
# Run immediately or schedule
```

### 4. Query Your Workspace
```bash
# With Databricks credentials configured:
curl http://localhost:8001/databricks/tables
curl http://localhost:8001/databricks/endpoints

# Get real metadata from your workspace
```

### 5. Check System Status
```bash
curl http://localhost:8001/health
curl http://localhost:8001/info
curl http://localhost:8001/databricks/status
```

## Performance Notes

### Latency
- **Workflow Start:** 1-2s (Supervisor analyzes)
- **Route Query:** 1-2s (Router decomposes)
- **Generate Question:** 0.5-1s (Interview)
- **Final Recommendation:** 2-3s (Synthesizer)
- **Databricks Queries:** 1-3s (API calls)

### Scalability
- **Current:** Single-threaded FastAPI (uvicorn)
- **For Scale:** Use gunicorn with multiple workers
  ```bash
  gunicorn -w 4 -b 0.0.0.0:8001 orchestration.server:app
  ```

### Optimization
- Gemini uses fast "Flash" model (cost-efficient)
- Async/await throughout for concurrency
- Databricks API calls cached when appropriate
- LangGraph state stored in memory (use Redis for scale)

## Troubleshooting Guide

### Service Won't Start
```bash
# Check Python syntax
python -m py_compile orchestration/*.py

# Check dependencies
pip list | grep gemini

# View full error
python orchestration/server.py (show traceback)
```

### Gemini API Errors
```bash
# Check API key
echo $GEMINI_API_KEY

# Test connectivity
curl https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=$GEMINI_API_KEY

# Check quota usage in GCP console
```

### Databricks Connection Issues
```bash
# Check credentials
cat orchestration/.env

# Test connectivity
curl -H "Authorization: Bearer $DATABRICKS_TOKEN" \
  "https://$DATABRICKS_HOST/api/2.1/serving-endpoints"

# Verify token hasn't expired (regenerate if needed)
```
